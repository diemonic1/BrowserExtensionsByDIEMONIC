Конфиги. В них хранится, как блокировать рекламу. После удаления они принудительно обновятся, когда вы попадёте на страницу, на которой блокируется реклама.

Последнее время проверки конфигов:

elementsToDelete

Эти элементы удаляются из страницы сразу, без каких-либо проверок.

#Селектор

elementsToCheckDelete

Эти элементы проверяются: если внутри встречается одно из banWords и нет ни одного stopWords — элемент удаляется.

#Селектор

elementsToCheckHide

Проверяются так же, как elementsToCheckDelete (через те же banWords/stopWords), но при совпадении не удаляются, а только скрываются (display: none) — используется для элементов внутри "живых" сеток, где удаление узла может сломать рендер страницы.

#Селектор

banWords

Общий список для elementsToCheckDelete и elementsToCheckHide: если внутри найденного элемента встречается любое из этих слов — он считается рекламой.

#Слово

stopWords

Общий список для elementsToCheckDelete и elementsToCheckHide: если встречается любое из этих слов — элемент не трогается, даже если совпал с banWords.

#Слово

elementsToHide

Эти элементы всегда только скрываются (display: none), без проверки banWords/stopWords.

#Селектор
